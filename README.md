# TOM – Your Smart Shopping Assistant

## Summary

TOM is an AI-powered shopping assistant that helps users plan their grocery trips efficiently. 
It suggests shopping lists based on habits, preferences, and what's already at home. 
Building AI course project.

## Background

The problem: People forget items, buy duplicates, or waste food because shopping is unplanned.
This happens to almost everyone, multiple times a week.
Personal motivation: A simple tool that thinks one step ahead — so you don't have to.

## Data and AI techniques

* User input: preferences, dietary habits, household size
* Historical purchase data (manual or app-connected)
* Techniques: collaborative filtering for recommendations, simple classification for categories

## How is it used

Used via mobile app before or during a shopping trip.
Target users: individuals, families, anyone who shops regularly.
Those affected: the user directly — saves time, reduces waste.

## Challenges

TOM does not replace human judgment about freshness or personal taste.
It cannot access real-time store inventory or prices (yet).
Privacy: purchase data is sensitive and must be handled carefully.

## What next

* Connect to supermarket APIs for live prices and availability
* Add expiration date tracking via photo input
* Expand to meal planning and recipe suggestions
